# calibre-server
Docker上で動かすcalibreサーバ

# 設定方法
1. サーバ上の任意のディレクトリ上にこのリポジトリをcloneする `git clone <このリポジトリ>`
2. `localhost:8080` にアクセスし、calibreの初期設定をする
3. calibreの本の保存先を `/library` に指定し、初期設定を完了させる
4. `localhost:8083` にアクセスし、calibre-webの初期設定をする
5. 認証情報を入力する。デフォルトでは右の通り。User: `admin`, Password: `admin123`
6. 本の保存先を `library` に指定する
